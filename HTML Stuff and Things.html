Font Styles

To write bold text, you will use the <b> tags.

A programmer may choose to instead write bold text with the <strong> tags. The difference being that the <strong> tag allows text to speech to add emphisis to words, where the <b> tag does not.

To write italic text, you will use the <i> tags.

A programmer may choose to instead write italic text with the <em> tags. The difference being that the <em> tag allows text to speech to add emphisis to words, where the <i> tag does not.

Super and Sub Script

Super Script is used typically when addressing dates, such as August 9th. By using the Super Script, we can make it appear as follows: August 9th.

Sub Script is used typically when addressing chemical compounds such as H2O, or water. By using the sub element, we can make it appear as follows: H2O.

It is important to note that both Super and Sub Script are commonly used in many forms of mathematics and can be required when dictating power notations, logrithms, or limits.
